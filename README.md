# Centro studio
Project for the course "Programmazione Web" 2025-26 at UniBO

### Project structure
```
.
└── src
    ├── api                 # API Endpoints
    ├── css
    ├── db                  # Database-related files
    ├── js
    ├── template            # Template files
    └── uploads             # Placeholder directory for files uploaded by users and static img
        ├── media           # directory for file and img uploaded by users
        └── static          # static img
```

### Instructions
Insert the Progetto-Tecnologie-Web folder into xampp/htdocs and create the database in file src/db/creatinonDB.sql.
